Mi último proyecto

Videogames App, es una aplicación que se alimenta de la API RAWG, y muestra las características principales de cada videogame, también se puede crear uno, a travez de un formulario controlado. Cuenta con una base de datos, implementada en Express y PostgreSQL, utilizando Sequelize, donde se guardan los llamados a la API y los videogames creados. Posee un searchbar, en donde se puede buscar por nombre y filtros por orden alfabético, género y rating. A su vez cuenta con un paginado que muestra determinada cantidad de datos por página. El Frontend fue realizado con React y Redux.
